# Step 5: Deliver

Finalize and present the output.

## Actions

1. Clean up scaffolding, debug output, temp files.
2. Place the final output where the user expects it.
3. Present a summary:
   - What was built (one sentence).
   - Constraints satisfied.
   - Tradeoffs or decisions made.

## Gate

Answer YES or NO:

- Is the output in its final location?
- Is the summary concise (under 5 sentences)?
- Are all temp artifacts cleaned up?

All YES = done.
