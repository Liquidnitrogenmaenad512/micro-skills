# Step 1: Scope

Extract exactly what is being asked before doing anything.

## Actions

1. Restate the request in one sentence.
2. List explicit constraints (format, length, language, dependencies, target path).
3. List implicit constraints (what the user probably expects but didn't say).
4. Identify inputs (files, data, context already in conversation).
5. Identify the deliverable (what exactly gets handed back, where it goes).

## Gate

Answer YES or NO:

- Can I state the deliverable in one sentence?
- Have I listed at least one explicit constraint?
- Do I know what inputs I'm working with?

All YES = proceed to Step 2.
Any NO = ask the user one clarifying question (max one).
