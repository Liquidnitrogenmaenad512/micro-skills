# Step 5: Deliver

Finalize and present the output.

## Actions

1. Clean up: remove debug output, temp files, console.logs.
2. Verify all files are in their correct locations per the plan.
3. Run the test suite one final time.
4. Present a summary:
   - Resource name and endpoints created
   - Files generated (with paths)
   - Test coverage achieved
   - Tradeoffs or decisions made

## Gate

- Is every generated file in its final location? YES/NO
- Is the summary under 5 sentences? YES/NO
- Do all tests pass? YES/NO

All YES = done.
